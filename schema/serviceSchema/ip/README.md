## Introduction

This repository contains the Mplify LSO Internet Protocol Service YAML payload schemas. The set of YAML
schemas to support Mplify W102 include:

- IP UNI
- IP UNI Access Link
- IP UNI Access Link Trunk
- IPVC End Point
- IPVC
- IP ENNI
- IP ENNI Link
- IP ENNI Common
- Virtual UNI
- IP Service Level Specification (SLS)

 - Alignment of data types used in Internet Protocol Service Schemas with Internet Protocol Product Schemas.

## Maturity Level

The schema files contained in this repository are evolving and subject to change. They are
based on documents that are either ratified standards or draft standards that
have not yet completed the review cycles and approvals necessary to achieve the
status as a Mplify standard. Mplify is making these publicly available at this time to
invite wider industry review.

- IP Service Attributes:
  - MEF 61.1 - **Published Standard**
- UNI Access Link Trunk, IP Addresses, and Mean Time to Repair Performance Metric Service
  - MEF 61.1.1 - **Published Standard**
